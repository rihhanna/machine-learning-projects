# House Price Prediction

A beginner Machine Learning project that predicts house prices using Multiple Linear Regression.

## Dataset Features

- House Size
- Number of Bedrooms
- Number of Bathrooms

## Target

- House Price

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Concepts Applied

- Multiple Linear Regression
- Model Training
- Prediction
- Data Exploration

## Example Prediction

Input:
- Size = 2100 sqft
- Bedrooms = 4
- Bathrooms = 3

Output:
- Predicted House Price
